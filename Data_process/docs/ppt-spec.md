# PPT 制作规范

> 本文件是 Claude Code 在 `Data_process` 项目下制作 PPT 的统一规范。
> 使用方式：对 Claude 说 **"按 ppt-spec 做一份关于 XX 的 PPT"** 即可。

---

## 一、PPT 结构规范（强制执行）

每份 PPT 必须严格遵循以下 5 段结构：

| 序号 | 段落 | 含义 | 必须包含 | 是否可省略 |
|------|------|------|----------|------------|
| 1 | **为什么** | 问题提出 / 背景 / 动机 | 研究对象、核心矛盾、为什么要做这个分析 | ❌ 不可省略 |
| 2 | **怎么做** | 方法论 / 计算路径 / 数据来源 | 理论公式、材料参数来源、几何假设、仿真工具 | ❌ 不可省略 |
| 3 | **结果** | 核心数据展示 | 关键数值表、趋势描述 | ❌ 不可省略 |
| 4 | **比较** | 参数化分析 / 方案对比 | 对比表格或对比图表（这是 PPT 的核心页，信息密度最高） | ❌ 不可省略 |
| 5 | **扩展** | 总结 / 展望 / 下一步 / 结论 | 选材建议、待补项、下一步计划 | ✅ 仅在明确需要时添加 |

### 段落放置规则

- 每个段落至少占 1 页 Slide
- "比较" 段如果内容过多，可以拆分为 2 页（如：参数化分析图 1 页 + 方案对比表 1 页）
- "扩展" 段默认不加——用户说"加上总结"或"写上结论"时才添加

## 二、格式约定

| 项目 | 约定 |
|------|------|
| **幻灯片尺寸** | 16:9 宽屏 (13.333 × 7.5 inches) |
| **语言** | 简体中文，技术术语保留标准英文缩写（如 k(T)、Qi、S21、NEP） |
| **封面** | 深色背景 (#1A1A1A) + 主题色装饰条 + 白字标题 + 灰字副标题 |
| **内页** | 白色背景 + 顶部主题色条 + 黑色标题 + 灰色副标题 |
| **字体** | Arial（英文/数字）+ 系统默认中文字体 |
| **页脚** | 右下角页码 |
| **主题色** | #1F77B4（蓝，主色） / #D62728（红，强调） / #2CA02C（绿） |

## 三、材料参数来源

极低温材料参数统一从 **NIST Cryogenics** 获取：

- **主站**: `https://trc.nist.gov/cryogenics/materials/`
- **G10 CR (⊥)**: `https://trc.nist.gov/cryogenics/materials/G-10%20CR%20Fiberglass%20Epoxy/G10CRFiberglassEpoxy_rev.htm`
- **G10 CR (∥)**: 同上页面，平行纤维方向
- **304 Stainless Steel**: `https://trc.nist.gov/cryogenics/materials/304StainlessSteel/304StainlessSteel_rev.htm`
- **OFHC Copper**: `https://trc.nist.gov/cryogenics/materials/OFHCCopper/OFHCCopper_rev.htm`

### NIST 数据拟合公式

**热导率** 使用 NIST 标准 log-polynomial：
```
log₁₀ k(T) = a + b·logT + c·(logT)² + d·(logT)³ + ... + i·(logT)⁸
```
其中 T 单位为 Kelvin，k 单位为 W/(m·K)。

**一维稳态热传导积分**：
```
Q = (A/L) × ∫_{T_cold}^{T_hot} k(T) dT

其中 K_int = ∫ k(T)dT，Q = (A/L) × K_int
```

**薄壁圆管截面积**：
```
A = π × D × t
D = 管径，t = 壁厚
```

## 四、Claude 执行流程

当用户说 "按 ppt-spec 做 PPT" 时，Claude 应按以下步骤执行：

1. **读取本规范** → 确认结构和格式
2. **确认任务范围** → 几页、什么主题、是否需要扩展段
3. **采集数据** → 从 NIST 获取材料参数、从已有 COMSOL 模型/数据文件读取仿真值
4. **理论计算** → 按规范公式计算（如需）
5. **生成图表** → matplotlib 出图，保存到 `output/` 目录
6. **组装 PPTX** → 用 python-pptx 逐页构建
7. **输出** → 保存到 `output/` 目录，报告路径

### 仿真数据规则

- 已有 COMSOL 结果的 → 直接引用
- 需要仿真但还没做的 → 表格中标注 "TBD" 或 "待仿真"，先以理论值占位
- "比较" 段应区分：理论值（实线/实心） vs 仿真值（虚线/空心），并标注
